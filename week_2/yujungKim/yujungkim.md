# * 저급언어 

<p> 저급 언어=기계 중심의 언어 = 기계가 직접 알아들을 수 있는 말
	컴퓨터는 전기로 동작함


	> 전기신호인 켜졌다(on),1 와 꺼졌다(off),0 2가지 상태 값으로 동작

	>0과 1로 표현되는 숫자를 우리는 2진수라고 말합니다.

	## 기계어(Machine Language) :2진수로 이뤄진 값으로 작성하는 프로그래밍 언어 

	-단점:  유지보수도 굉장히 어렵고 사용하기 어려움 


	기계어 - *컴파일러* -> 어셈블리어 


	* **컴파일러(COMPILER)** :기계가 바로 고급언어를 알아들을 수 없기 때문에 고급언어로 작성된 문장을 저급언어로 바꿔야 하는 과정에서 사용되는 도구

</p>

# * 고급 언어 : 사람 중심의 언어로 사람이 좀 더 이해하기 쉬운 문법으로 프로그래밍 함

*최근 컴퓨터 성능이 좋아지면서 컴파일러와 관련된 기술이 발전하면서 사람 중심의 언어로 프로그래밍할 수 있어짐*

	-자주 사용되는 프로그래밍 언어 

 		-C : 1972년 미국 벨 연구소의 데니스 리치에 의해 개발된 고급 언어로써 
 		**시스템 프로그래밍**에 가장 적합한 평가를 받는 언어


 		-Swift : 2014년 WWDC(Apple WorldWide Developers Conference)에서 공개한 프로그래밍 언어. 최근에 만들어진 언어로 현대 프로그래밍 언어의 발전을 대다수 계승한 모던 프로그래밍 언어라고 말할 수 있습니다.
 		ios 코딩시 사용됨. 

 		-Kotlin : IntelliJ IDEA의 개발사 JetBrains에서 2011년에 개발한 프로그래밍 언어
 		JVM기반의 언어이며 Java와의 상호 운영이 100% 지원 
 		Swift와 마찬가지로 현대 프로그래밍 언어의 발전을 대다수 계승한 모던 프로그래밍 언어임. 
 		and 코딩시 사용됨.

 		-Python : 프로그래밍 입문자가 읽기 쉽고 적은 코드를 사용하여 프로그램을 개발가능 많은 사람에게 추천되는 언어이며, 데이터 과학에서도 자주 사용되며 웹사이트 개발에서도 많이 사용됨. 최근 python은 ML (machine learning)에서도 많이 사용.

		-JAVA : 1995년 썬 마이크로 시스템즈에서 개발한 객체지향 프로그래밍 언어 거의 매년 세계에서 가장 많이 사용되는 인기 1등을 차지

# 웹 관련 인기 언어

* Python : 프로그래밍 입문자가 읽기 쉽고 적은 코드를 사용하여 프로그램을 개발할 수 있습니다. 
많은 사람에게 추천되는 언어이며, 데이터 과학에서도 자주 사용되며 웹사이트 개발에서도 많이 사용되고 있습니다. 
--> 주로 대학원 연구생들이 딥러닝 작업할때 사용 

* PHP : 웹의 80% 이상이 PHP로 만들어졌다고 말합니다. 그만큼 PHP는 웹 개발에서 많이 사용됩니다. 
 공기업을 제외한 대부분의 중소, 중견 회사에서 사용. 
공기업에서는 jsp라는 언어 사용 


* JavaScript : 자바 스크립트는 처음 시작이 브라우저에서 동작하는 언어였습니다. 현재는 서버에서도 작성하는 프로그램(nodejs)으로 점차 영역을 넓혀가고 있습니다.
 프론트 개발자라면 반드시 알아야 할 언어입니다. 자바스크립트 커뮤니티도 점점 더 거대해지고 있습니다.
 
* JAVA : 엔터프라이즈 소프트웨어 환경에 잘 맞는 언어입니다. 큰 규모의 소프트웨어 개발에 자바언어가 많이 사용되고 있습니다. 
JAVA언어를 지원하는 수많은 커뮤니티에 위해서, 지속적으로 발전되어 훌륭한 구조와 설계 기법들이 잘 갖춰져 있습니다.

* Ruby : 빠른 개발에 널리 사용되며, 단순함과 세련된 웹 어플리케이션을 만들 수 있기 때문에 인기 있는 언어 중의 하나입니다.
그러나 점점 사용되지 않고 있음.
