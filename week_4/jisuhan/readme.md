## JS Comparison

```
== (equal)
=== (strict equal)
!= (not equal)
!== (strict not equal)
> (greater than)
< (less than)
>= (greater than or equal to)
<= (less than or equal to)
```

여기서 체크하고 넘어가야할 부분은 == 대신 ===을 쓰고 != 대신 !==를 쓰는 것이 좋다 정도입니다

## JS Conditional Expression

if, else if, else

```
if(true){
  console.log('logging')
}
//logging
```

조건이 들어가는 소괄호()에 들어가는 조건이 true면 중괄호{} 안에 코드 실행

```
if(condition){
  console.log(something)
} else if(condition){
  console.log(else)
} else {
  console.log(blah)
}
```
